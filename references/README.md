# references/

Dieser Ordner ist nur für öffentliche, repo-spezifische Referenzen gedacht.

Nicht committen:

- interne Hostnames, private Domains, IP-Pläne, VLAN-Tabellen oder Topologie-Auszüge
- personenbezogene Daten
- Secrets, Tokens, Cookies, Recovery Codes, Private Keys oder Zugangsdaten
- private Inventare, lokale Budgetzustände oder operative Notizen

Für private Kontextdaten nutze lokal einen ignorierten Pfad wie `references/local/` oder `LOCAL_CONTEXT.md`.
