# Data analysis project

## Steps

1. Identify datasets and file formats.
2. Run or propose a data quality audit before conclusions.
3. Keep raw data immutable.
4. Write reproducible scripts, not one-off mystery calculations.
5. Label units, filters, date ranges, and assumptions.
6. Separate observed facts from interpretation.

## Deliverables

- `reports/` for narrative outputs.
- `outputs/` for generated tables/charts.
- `scripts/` for reproducible analysis.
- Data quality note in every result summary.
