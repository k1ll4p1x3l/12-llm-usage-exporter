# Writing project

## Steps

1. Identify audience, purpose, tone, length, and output format.
2. Create an outline before drafting long text.
3. Maintain consistency files where useful: `docs/STYLE_GUIDE.md`, `docs/GLOSSARY.md`, `docs/OUTLINE.md`.
4. Work chapter-by-chapter or section-by-section.
5. Do not copy protected text. Summarize or create original prose.
6. Mark factual claims that need research.

## Output

- Outline or edited text.
- Change notes.
- Consistency issues.
- Next section recommendation.
